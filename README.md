# analise-vendas-ecommerce
Projeto de análise exploratória de vendas de e-commerce utilizando Excel.


## **Análise de Vendas de E-commerce**

**Sobre o Projeto**

Este projeto foi desenvolvido com o objetivo de analisar dados de vendas de um e-commerce e identificar padrões de faturamento e lucratividade entre categorias de produtos e regiões.


A análise foi realizada utilizando Microsoft Excel, com foco em exploração de dados, tabelas dinâmicas e geração de insights de negócio.


**Objetivo**

Analisar dados de vendas de um e-commerce para identificar:
Categorias mais lucrativas;

Regiões com maior lucratividade;

Categorias com maior faturamento.


**Perguntas de Negócio**

Qual categoria gera mais lucro?

Qual região gera mais lucro?

Qual categoria gera mais faturamento?


**Ferramentas Utilizadas**

Microsoft Excel

Tabelas Dinâmicas

Gráficos Dinâmicos


**Metodologia**

Importação da base de dados.

Tratamento e organização dos dados.

Construção de tabelas dinâmicas.

Criação de gráficos para análise.


## *Identificação dos principais insights.*

📈 **Categoria mais lucrativa**

**Furniture (Móveis)** apresentou o maior lucro acumulado da base analisada.

🌎 **Região mais lucrativa**

**North (Norte)** apresentou o maior lucro acumulado entre todas as regiões.

💰 **Categoria com maior faturamento**

**Home Decor (Decoração de casa)** apresentou o maior faturamento total da base.

🎯 **Insight de negócio**

Embora Home Decor tenha apresentado o maior faturamento, Furniture foi a categoria mais lucrativa, demonstrando que faturamento e lucro não necessariamente possuem a mesma liderança.

Durante o desenvolvimento, enfrentei desafios relacionados à importação e tratamento dos dados, incluindo correção de formatação, tipos de dados e organização da base para análise.
